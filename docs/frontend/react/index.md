# React 开发入门

React 前端框架是当下常用的网页前端框架，基于 JavaScript 语言，并提供了对 TypeScript 语言的支持。

## 前置知识

- 具有 JavaScript 以及 TypeScript 语言基础

- 理解 HTML 标签语法和 CSS 语法

- 有简单的状态机思维

## 后续拓展

React 作为广泛运用的网页前端框架，基本上可以应对大部分常见的网页前端编写任务。
